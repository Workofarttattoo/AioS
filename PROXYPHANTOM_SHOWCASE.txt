├───────────────────────────────────────────────────────────────────────────┤
│                                                                            │
│  1️⃣  SNIPER                                                               │
│      → One payload position at a time                                     │
│      → Use case: Parameter testing, SQL injection                         │
│                                                                            │
│  2️⃣  BATTERING RAM                                                        │
│      → Same payload in all positions                                      │
│      → Use case: Password guessing, authentication bypass                 │
│                                                                            │
│  3️⃣  PITCHFORK                                                            │
│      → Different payloads per position (zipped)                           │
│      → Use case: Credential stuffing, username:password pairs             │
│                                                                            │
│  4️⃣  CLUSTER BOMB                                                         │
│      → All combinations (cartesian product)                               │
│      → Use case: Brute force, exhaustive testing                          │
