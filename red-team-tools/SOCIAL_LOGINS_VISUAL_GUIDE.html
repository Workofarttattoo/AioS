üöÄ Go to Google Cloud Console

  • Click Create

  • In sidebar: "APIs & Services" ‚Üí "Library"

    https://trokobwiphidmrmhwkni.supabase.co/auth/v1/callback
  • Click Create
  • Copy Client ID and Client Secret

Step 5: Configure in Supabase

Back in Supabase:

Step 1: Create Service ID

üöÄ Go to Apple Developer
  • Click "+" to add identifier
  • Select "Services IDs", click Continue
  • Fill in:
    Description: Red Team Tools
    Identifier: com.corporationoflight.redteamtools
  • Click Continue, then Register

Step 2: Configure Sign In with Apple

  • Click on the Service ID you created
  • Check "Sign In with Apple"
  • Click Configure
  • Fill in:
    Domains: red-team-tools.aios.is
    Return URLs: https://trokobwiphidmrmhwkni.supabase.co/auth/v1/callback
  • Click Save, Continue, Save

Step 3: Create Private Key

üîë Go to Keys
  • Click "+"
  • Key Name: Red Team Tools Auth Key
  • Check "Sign In with Apple", click Configure
  • Select your Service ID, click Save
  • Click Continue, then Register
  • Download the .p8 file (only shown once!)
  • Note the Key ID

Step 4: Get Team ID

  • Go to Apple Developer Account
  • Your Team ID is in top right corner

Step 5: Configure in Supabase

Back in Supabase:

  • Find Apple provider
  • Toggle it ON
  • Fill in:
    Services ID: com.corporationoflight.redteamtools
    Team ID: [from step 4]
    Key ID: [from step 3]
    Private Key: [open .p8 file, copy entire contents]
  • Click Save
