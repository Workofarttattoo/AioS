üöÄ Quick Start Guide

1. Start the Backend API

cd /Users/noone/aios/docs/api
pip install -r requirements.txt
python security_tools_api.py

2. Test the API

# Health check
curl http://localhost:5000/api/health

# Test SQLgps scan
curl -X POST http://localhost:5000/api/sqlgps/scan \
-H "X-API-Key: demo_key_12345" \
-H "Content-Type: application/json" \
-d '{"url": "http://localhost/test", "techniques": ["boolean"]}'

3. Open the Frontend

open file:///Users/noone/aios/docs/index.html

üåê Production Deployment

Option 1: Gunicorn

Production WSGI server with 4 worker processes:

gunicorn -w 4 -b 0.0.0.0:5000 security_tools_api:app

Option 2: Docker

Containerized deployment with Docker Compose:

docker build -t security-api .
docker run -p 5000:5000 security-api

Option 3: Nginx + Gunicorn

Full production stack with reverse proxy and SSL:

systemctl start security-api
systemctl reload nginx

Monitoring & Logs

Production logging and monitoring setup:

tail -f /var/log/security-api/api.log

See DEPLOYMENT_GUIDE.md for complete production deployment instructions.
