📂 DirReaper

Web directory and file enumeration with intelligent fuzzing

Overview

DirReaper is a high-performance directory enumeration tool for discovering hidden paths, backup files, and sensitive directories on web servers during authorized security assessments.

Features

  • 📂 Intelligent directory fuzzing
  • 🔍 Recursive path discovery
  • 📝 Custom wordlist support
  • ⚡ Multi-threaded scanning
  • 🎯 Status code filtering (200, 301, 403, etc.)
  • 📊 Real-time progress tracking
  • 💾 Export results (JSON, CSV, HTML)

Quick Start

# Basic directory scan python -m red-team-tools.dirreaper --url https://example.com --wordlist common.txt # With JSON output python -m red-team-tools.dirreaper --url https://example.com --wordlist common.txt --json # Health check python -m red-team-tools.dirreaper --health --json

Common Wordlists

  • common.txt - Top 1000 paths
  • comprehensive.txt - 10K+ paths
  • api-endpoints.txt - REST API paths
  • backup-files.txt - Common backup patterns
