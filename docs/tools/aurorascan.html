🔍 AuroraScan

Advanced network reconnaissance and intelligent host discovery

Overview

AuroraScan is a ParrotOS-inspired network reconnaissance tool designed for defensive security assessments. It provides intelligent host discovery, service enumeration, and network mapping capabilities.

Features

  • 🔍 Intelligent host discovery
  • 📊 Service enumeration
  • 🎯 Multiple scan profiles (stealth, balanced, aggressive)
  • 📈 JSON output for automation
  • 🖥️ GUI interface available
  • ✅ Health check support
  • 🔒 Demo mode for safe testing

Quick Start

# Basic network scan python -m red-team-tools.aurorascan 192.168.1.0/24 --profile recon --json # Demo mode (safe testing) python -m red-team-tools.aurorascan --demo --json # GUI mode python -m red-team-tools.aurorascan --gui # Health check python -m red-team-tools.aurorascan --health --json

Scan Profiles

  • stealth: Slow, low-noise scanning
  • recon: Balanced reconnaissance
  • full: Comprehensive deep scan
  • quick: Fast discovery scan
