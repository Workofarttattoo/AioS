👻 ProxyPhantom

HTTP/S proxy manipulation and traffic interception analysis

Overview

ProxyPhantom is an intelligent HTTP/S proxy tool for intercepting, analyzing, and manipulating web traffic during security assessments. It provides deep packet inspection and request/response modification capabilities.

Features

  • 👻 Transparent HTTP/S interception
  • 🔍 Deep packet inspection and analysis
  • ✏️ Request/response modification
  • 📊 Traffic pattern analysis
  • 🔐 SSL/TLS certificate management
  • 💾 Session recording and replay
  • 🎯 Fuzzing and injection testing

Quick Start

# Start proxy on default port (8080) python -m red-team-tools.proxyphantom --port 8080 # Health check python -m red-team-tools.proxyphantom --health --json

Use Cases

  • API security testing
  • Session management analysis
  • Authentication bypass testing
  • Input validation fuzzing
  • SSL/TLS configuration audit
