# AgentaOS Documentation

This directory contains the core documentation for the AgentaOS (AI:OS) project, structured according to the BMAD methodology.

## Strategic Documents
- [Product Brief](product-brief.md): High-level vision and problem statement.
- [Product Requirements Document (PRD)](PRD.md): Detailed functional and non-functional requirements.

## Technical Documents
- [Architecture](architecture.md): System design, component overview, and technology stack.
- [UX Design](ux-design.md): User interaction principles, interfaces, and journeys.
- [Project Context](project-context.md): Current state analysis, including file structure and known issues.

## Planning & Execution
- [Epics and Stories](epics-and-stories.md): Roadmap of critical tasks ("Tickets") to stabilize and enhance the project.

## Methodology
These documents were created retroactively to align the existing codebase with the BMAD (Build, Measure, Analyze, Design) framework, ensuring the project remains viable and maintainable.
